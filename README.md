# Spring Boot Projects

## 📌 About the Repository

This repository contains the projects, examples, and exercises completed while learning and practicing with Spring Boot. The primary goal of these projects is to understand backend development, REST API creation, and database integration using modern Java development practices.

## 🚀 Technologies Used

* Spring Boot
* Java>
* Spring MVC
* Spring Data JPA
* REST APIs
* MySQL
* Maven
* Hibernate
* Postman

## 📂 Features Implemented

* Project setup using Spring Initializr
* Dependency Management with Maven
* REST API Development
* CRUD Operations
* Database Connectivity
* Entity and Repository Layer
* Service Layer Implementation
* Exception Handling
* Request Validation
* JSON Request and Response Handling


## 🎯 Learning Outcomes

Through these projects, I gained practical experience in:

* Building RESTful APIs
* Implementing layered architecture
* Connecting applications with databases
* Handling exceptions and validations
* Managing dependencies using Maven
* Understanding backend application development workflows
